# 💰 Price Monitor in Python

Monitor de preços desenvolvido em Python que permite acompanhar automaticamente o valor de um produto em um e-commerce. O script realiza verificações periódicas, registra o histórico de preços e alerta quando o valor alvo é atingido.

## 🚀 Funcionalidades
- Monitoramento automático de preços de um produto específico
- Definição de preço alvo para alerta de oportunidade
- Registro do histórico de preços em arquivo `.txt`
- Logs com data e hora de cada verificação
- Execução contínua com intervalo configurável

## 🛠 Tecnologias utilizadas
- Python 3
- Requests
- BeautifulSoup4
- Datetime
- Time

## 📦 Como executar
1. Clone o repositório:
```bash
git clone https://github.com/MatheusDevlog/automated-price-tracker.git
```

2. Instale as dependências:
```bash
pip install requests beautifulsoup4
```

3. Execute o arquivo:
```bash
automated-price-tracker.py
```

## 📌 Observações
- Projeto desenvolvido para fins de estudo e portfólio
- A estrutura do site monitorado pode mudar, exigindo ajustes no código
- Recomendado utilizar com moderação para evitar bloqueios
